

# DALL-E Image Generation with .NET 8

## Overview

This project demonstrates the integration of DALL-E, a cutting-edge AI model by OpenAI, with .NET 8 to generate images from textual descriptions. DALL-E's powerful capabilities allow for creative and innovative applications in various fields.

## Features

- Generate images from text descriptions using DALL-E
- Seamless integration with .NET 8
- Sample code and detailed instructions included

## Prerequisites

- .NET 8 SDK installed
- OpenAI API key

## Getting Started

1. Clone the Repository**
   ```sh
   git clone https://github.com/Tayyab94/DALL-E-Image-Generation.git
   cd DALL-E-Image-Generation
   ```

2. **Install Dependencies**
   ```sh
   dotnet restore
   ```

3. **Set Up OpenAI API Key**
   - Replace `YOUR_API_KEY` in `appsettings.json` with your OpenAI API key.

4. **Run the Application**
   ```sh
   dotnet run
   ```

5. **Generate Images**
   - Use the provided interface to input text descriptions and generate images.
